## Technique 3: Six Thinking Hats 六頂思考帽

### ⚪ 白帽：事實

- 全球睡眠產業 2025 年市場規模約 $1,000 億美金
- 台灣約 1/4 人口有睡眠困擾（約 600 萬人）
- 華語市場「台灣在地」缺口存在
- 睡眠 App 平均 30 天留存率約 10-15%

### ❤️ 紅帽：情感

- 用戶想要的體驗：被溫柔照顧 + 簡單有效 + 跟人連結
- 核心情感不是「好玩」，是「被在乎」

### 🌟 黃帽：優勢

- **主打優勢**：台灣在地 + 不焦慮設計
- 口號概念：「來自台灣，為你的心而設計」

### ⚫ 黑帽：風險

- **最擔心**：AI 技術門檻、留存率魔咒
- **應對**：MVP 先用規則模擬；驚喜+實體箱+社群三重綁定

## Deep Dive: Feature Specifications 功能深入探討

### #36 一鍵入睡 One-Tap Sleep

> 整個 App 只有一個按鈕，AI 自動判斷一切

**核心理念：** 極簡主義 × 智慧自動化

**功能規格：**

| 項目            | 說明                                                 |
| --------------- | ---------------------------------------------------- |
| **主介面**      | 全螢幕單一按鈕，背景隨時間漸變（日落色調）           |
| **AI 判斷邏輯** | 綜合時間、過往習慣、當日活動、心率等自動選擇最適方案 |
| **情境模式**    | 平日晚安 / 週末放鬆 / 出差調適 / 焦慮安撫            |
| **學習機制**    | 追蹤用戶行為，持續優化推薦                           |
| **退出機制**    | 輕觸螢幕任意處可進入手動調整                         |

**用戶旅程：**

1. 打開 App → 看到一個溫暖的圓形按鈕
2. 按下按鈕 → AI 自動啟動今晚最適合的入睡流程
3. 躺下 → 聲音、燈光（如有整合）自動調節
4. 入睡 → App 自動淡出

**技術需求：**

- 行為學習引擎
- 時間/日期/天氣 API
- 可選：穿戴裝置整合

### #10 雙模態陪伴系統 Dual-Mode Companion

> Widget 被動陪伴 + 語音關鍵時刻介入

**核心理念：** 不打擾的存在感 + 適時的溫暖介入

**功能規格：**

| 模態                | 觸發條件                       | 呈現方式                        |
| ------------------- | ------------------------------ | ------------------------------- |
| **Widget 被動模式** | 常駐桌面                       | 一句話 + 漸變背景色，每小時更新 |
| **語音主動模式**    | 睡前時段 / 偵測焦慮 / 用戶呼叫 | 溫柔語音，像朋友聊天            |

**Widget 內容範例：**

- 「今天辛苦了，晚上記得對自己好一點」
- 「窗外的雨聲也許是最好的白噪音」
- 「還有 2 小時是你的睡前時光」

- 睡前 30 分鐘：「嘿，我注意到你還在滑手機，要不要一起準備睡覺？」

- 失眠時：「睡不著沒關係，我陪你」

**技術需求：**

- iOS/Android Widget SDK
- 情緒偵測演算法（可選：整合穿戴裝置）
- TTS 語音合成（台灣口音）
